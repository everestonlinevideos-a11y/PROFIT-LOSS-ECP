Price at which article is bought. All % on CP.

📈 Markup

⚡ Core Formulas — ECP Method
✅ Gain %
Gain = SP − CP
Gain% = Gain/CP × 100
❌ Loss %
Loss = CP − SP
Loss% = Loss/CP × 100
🏷️ Discount %
Discount = MP − SP
Disc% = Disc/MP × 100
📊 Markup %
Markup% = (MP−CP)/CP×100
Always on CP. SP always on CP!
🔑 CP/MP Relation
CP/MP = (100−d)/(100+g)
d = discount%, g = gain%
📉 Same SP Loss Formula
Loss% = x²/100
When gain x% and lose x% on same SP
🔥 Golden Trick — CP:SP Ratio → Instant %
CP:SP = 3:4 → Gain = 1, CP = 3 → Gain% = 1/3×100 = 33.33%
CP:SP = 4:3 → Loss = 1, CP = 4 → Loss% = 1/4×100 = 25%
CP:SP = 5:8 → Gain = 3, CP = 5 → Gain% = 3/5×100 = 60%
CP:SP = 8:9 → Gain = 1, CP = 8 → Gain% = 1/8×100 = 12.5%
⚡ Successive Discounts Formula
Two discounts x%, y%: Single equivalent = x+y − xy/100
Example: 30% & 20% → 30+20 − 30×20/100 = 50−6 = 44%

Buy n get m free: Equivalent discount = m/(n+m) × 100%
Buy 5 get 3 free: 3/8 × 100 = 37.5%
